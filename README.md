# Campaign Notes

Built using Obsidian.md. Files can be referenced and even edited from github,
but indexing, graph associations, and timeline won't be visible.

## Where to look

* `[assets]` contains both attachments (images), and small template files used for content creation.
* `references` contains general DM/GM references and notes gathered from other places
* [Plot Threads](PlotThreads.md)
* [Questions And Answers](Questions%20And%20Answers.md)

### Compendium
* Most linkable notes, including summaries of NPCs and locations, are in the compendium folder. 
* If you are using Obsidian, the [Index](Index.md) will render a grouped listing of items in the compendium.
* The structure is _fairly_ flat, with some scoping:
    * NPCs or locations that are scoped are grouped, e.g. `phandalin/` contains all NPCs and locations that are based solely in the town.
    * `groups` contains pages for faction, merchant, or mercenary groups
    * `statblocks` contains monster statblocks or related information pages

### Logs
* [logs/Game Time](logs/Game%20Time.md) -- the old single-page list from trello
* The logs directory contains a running player-centric view of what has happened. 
    * It should not reference `compendium` content
    * It may reference `writing`
* Files are numbered based on the number of days that have passed. If an entry spans several days, the name should have some indication of included range.
* Each log entry will reference [time](reference/time.md) ... eventually. Not quite there yet.

### Writing
* Past session write-ups or fill-in entries
* Should not link or reference `compendium` content (player-friendly)
* May reference `logs`
